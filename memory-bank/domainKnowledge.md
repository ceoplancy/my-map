# Domain Knowledge — 앤트리맵 (EntryMap)

> 주주·주식 관리 서비스 → B2B SaaS 플랫폼 도메인

---

## 현재 서비스 (단일 회사)

| 용어 | 설명 |
|------|------|
| **주주명부** | 엑셀 업로드로 관리하는 주주 목록 (주소, 주식 수, 상태 등) |
| **지도** | 카카오맵 기반 주주 위치 마커 (미방문/보류/완료/실패 등 상태) |
| **용역** | 현장 요원 — 주주 방문·상태 변경·메모·사진(위임장/신분증) 업로드 |
| **관리자** | 주주명부 업로드, 사용자(용역) 관리, 엑셀 import/export |

---

## 플랫폼화 후 (목표)

| 용어 | 설명 |
|------|------|
| **상장사** | 상장 회사 — 플랫폼 가입 후 자신의 주주명부·용역을 관리 |
| **의결권 대행사** | 의결권 대리 업무 수행 — 동일 플랫폼 UI로 복수 상장사 주주명부 관리 가능 |
| **워크스페이스** | 계정별 독립 공간 (복수 주주명부, 용역, 대시보드) |
| **운영사** | 앤트리맵 최고 관리자 — 가입 승인/반려, 용역 충원 요청 접수 |
| **활성화 기간** | 주주명부별 노출/수정 가능 기간 (설정·협의 중) |

---

## 기술 참고

- **DB**: Supabase (PostgreSQL), 타입 생성 → `src/types/db.ts`
- **현재 테이블**: `excel` (주주 레코드: address, company, status, stocks, history, image 등)
- **스택**: Next.js (Pages Router), Supabase, Kakao Maps, Sentry
